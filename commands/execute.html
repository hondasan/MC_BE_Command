🎯 /execute コマンド

「モブが〜したら」「だれがどこで」など、コマンドを動かす人や場所、条件を決める最強のコマンドだよ！

🔍 /execute ってなに？

普通のコマンドは「実行したプレイヤー本人」が「今いる場所」で動きます。

しかし、/executeを使うと、「だれが(as)」「どこで(at)」「どんな条件で(if)」コマンドを実行するかを自由自在に指定できます。

新しい書き方（新構文）に対応しているよ！
統合版のバージョン1.19.50から、書き方が新しくなりました。このページでは最新の書き方を分かりやすく説明しています。

🛠️ よく使うサブコマンド

1. as（だれとして実行するか）

コマンドの「実行者」をチェンジします。これを使うと、モブ自身に自分の名前でコマンドを動かさせることができます。

2. at（どこで実行するか）

コマンドが動く「基準の場所」を指定したモブやプレイヤーの場所にチェンジします。

3. as と at の組み合わせ

「ゾンビとして、ゾンビのいる場所で」のように組み合わせて使うことが多いです。

❓ 条件（もし〜なら実行する）

if を使うと、「もし○○だったらコマンドを実行する」という条件付きの仕掛けを作ることができます。

1. if block（もしそこに特定のブロックがあったら）

2. if entity（もし特定のモブが近くにいたら）
