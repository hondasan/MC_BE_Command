🎮 ゲームルールのコマンド

「死んでもアイテムを落とさない」「クリーパーで地面が壊れない」など、ワールドの特別なルールを変えるコマンドだよ！

🎮 /gamerule（ゲームルールを変更する）

ゲーム内で設定画面を開かなくても、コマンドひとつで特別なルールに変更できます。

■ コマンドのかたち

/gamerule <ルール名> <値(true または false または 数値)>
・<ルール名> : 変更したい設定の英語キー名（例: keepInventory）
・<値> : true (有効・オン) または false (無効・オフ) または 数字

■ とくによく使うおすすめの設定

📜 ゲームルール設定キー一覧 (タップでコピー)

変更したいルールのカードをタップすると、その設定を切り替えるコマンドがコピーできるよ！
