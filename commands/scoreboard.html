📊 スコア・表示のコマンド

ゲームの得点をかぞえたり、画面に大きな文字を出すコマンドだよ！ミニゲームづくりに大活躍するよ！

📊 /scoreboard（点数を記録する・表示する）

プレイヤーごとに得点（スコア）を記録し、それを画面上に表示させることができます。

■ コマンドの動かし方（3ステップ）

ステップ 1 : スコアの項目（ボード）を作る
/scoreboard objectives add <ボード名> dummy [画面に出す名前]
※ dummyは「コマンドでしか点数が変わらない」という意味です。
ステップ 2 : 画面にスコアを表示する
/scoreboard objectives setdisplay <場所> <ボード名>
※ <場所> : sidebar (画面の右側), list (一時停止メニュー), belowname (プレイヤーの頭の上)
ステップ 3 : 点数を増やす・減らす
/scoreboard players add <プレイヤー> <ボード名> <増やす点数>
/scoreboard players remove <プレイヤー> <ボード名> <減らす点数>
/scoreboard players set <プレイヤー> <ボード名> <指定の点数>

🏷️ /tag（ターゲットに印・タグをつける）

プレイヤーやモブに「目印（タグ）」をつけることができます。これを使うと、「鬼ごっこの鬼チーム」「逃げるチーム」などをコマンドで分類できます。

■ つかいかたの例

📺 /title（画面に大文字を表示する）

画面のど真ん中に、映画やイベントのような大きな文字を表示します。ミニゲームのスタート合図やゴール時に使います。
※文字に色をつけたいときは、文字の前に §c（赤色）や §e（黄色）などの色コード（セクション記号）を入れてください。
